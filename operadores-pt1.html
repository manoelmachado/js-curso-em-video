Operadores parte 1

O JS possui várias famílias de operadores, mas nas aulas iremos focar nos operadores aritméticos, operadores de atribuição, relacionais, lógicos e ternário.

Nessa aula, iremos focar no aritméticos e operadores de atribuição.

Operadores Aritméticos:

São operadores que são usados para fazer cálculos:

  • + = soma
  • - = diminuição
  • * = multiplicação
  • / = divisão (números reais)
  • % = divisão (números inteiros/resto da divisão)
  • ** = potência (n ao quadrado)

Ordem de precedência:

Em todas as linguagens de programação, os operadores tem ordem de precedência, seguindo sempre essa ordem:

  • ( ) o operador vai ler primemramente o que está dentro dos parenteses
  • ** após isso, ele vai ler o operador de potência
  • * / % logo após ele vai ler os operadores de multiplicação, divisão e resto da divisão, lembrando que sempre ele irá ler da esquerda para a direita.
  • + - e por final, ele vai ler os operadores de soma e diminuição, também seguindo a regra de ler da esquerda para a direita.

Atribuição simples:

Podemos guardar os resultados dessas somas em algumas variáveis e fazer mais variáveis em sequência, exemplo:

  • var a = 5 + 3 ou seja, var a = 8
  • var b = a % 5 ou seja, var b = 3
  • var c = 5 * b ** a ou seja, var c = 45
  • var d = 10 - a / 2 ou seja, var d = 6
  • var e = 6 * 2 / d ou seja, var e = 2
  • var f = b % e + 4 / e, ou seja var f = 3

Lembrando que o JS sempre irá seguir a ordem de precedência na hora de fazer os cálculos, como está em destaque acima.

Auto-atribuições e suas simplificações:

Auto-atribuições são atribuições à própria variável, como o exemplo a seguir:

  1. var n = 3
  2. n = n + 4, ou seja, n = 7
  3. n = n - 5, ou seja, n = 2
  4. n = n * 4, ou seja n = 8
  5. n = n / 2, ou seja n = 4
  6. n = n ** 2, ou seja n = 16
  7. n = n % 5, ou seja n = 1

Simplificando as auto-atribuições:

  • n = n + 4, simplificando n += 4
  • n = n - 5, simplificando n -= 5
  • n = n * 4, simplificando n *= 4
  • n = n / 2, simplificando n /= 2
  • n = n ** 2, simplificando n **= 2
  • n = n % 5, simplificando n %= 5

Obs.: a grande maioria das linguagens de programação aceitam as variações acima.
