Variáveis

No javascript moderno, além de utilizar a palavra var, também podemos usar a palavra let.

Exemplo:

  • var n1 = 5
  • var n2 = 8.5
  • var n3 = 15
  • var s1 = "JavaScript"
  • var s2 = 'Curso em Vídeo'
  • var s3 = `Guanabara`

OBS.: SEMPRE ao ler o sinal de igual =, lê-se no JavaScript como RECEBE. n1 recebe 5, n2 recebe 8.5, n3 recebe 15.

Identificadores de variáveis:

Os identificadores das variáveis vem após o 'var', e existem algumas regras para eles, sendo essas regras:

  • Podem começar com letras, $ ou _
  • Não podem começar com números
  • É possível usar letras ou números
  • É possível usar acentos e símbolos
  • Não podem conter espaços
  • Não podem ser palavras reservadas

Dicas para criação de variáveis:

Letras maiúsculas e minúsculas fazem diferenças. Ao contrar uma variável b1 por exemplo, use sempre o b minúsculo. Você pode ter uma variável b1 e uma B1.

Tente escolher nomes coerentes para as variáveis, como exemplo var idade = 18, var sexo = 'masculino', var nome = 'Manoel'

Evite se tornar um programador alfabeto ou um programador contador, assim como citado acima, muitos programadores usam variáveis de letra e número, como por exeplo: var a1 = 18, var a2 = 'masculino', var a3 = 'Manoel', use nomes coerentes na hora de criar os identificadores de variáveis.

Para que serve as variáveis?

As variáveis são utilizadas para armazenar dados e no começo desse curso iremos focar nos tipos de dados number, string e boolean

Mas também temos diversos outros tipos de dados primitivos, tal quais como:

  • number // Infinity, NaN
  • string
  • boolean (boleano)
  • null
  • undefined
  • object // Array
  • function

Para identificar os dados, você pode usar o comando typeof no terminal
