Manipulação e tratamento de dados com JavaScript

Formatando números:

Na aula passada criamos alguns prompt que apareceu em janela ao abrir o navegador, mas ao inserir os dados não acontecia nada, nessa aula aprenderemos a manipular esses dados.

O window.prompt retorna uma string, mesmo que você digite um número, é uma carecterística do comando. E ao usar o símbolo + ele pode te retornar dois valores, dependendo do tipo do dado. No caso de string, ele concatena (exemplo na aula anterior, onde concatenamos a frase + nome), e em caso de números, ele faz a adição.

Mas como faríamos uma soma dos números nessa página? Nesse caso, faremos uma conversão de string para número, desse jeito o símbolo + vai ser para adição.

Como a conversão de string para número é feita? Existem algumas maneiras de realizar essa conversão. A primeira delas é usar o comando Number.parseInt(n) - para números inteiros, e Number.parseFloat(n) para números reais.

A variavel anteriormente estava da seguinte maneira:

var n1 = window.prompt('Digite um número: ')

var n2 = window.prompt('Digite outro número: ')

E após adicionar o Number.parseInt(n) ficou assim:

var n1 = Number.parseInt(window.prompt('Digite um número: '))

var n2 = Number.parseInt(window.prompt('Digite outro número: '))


A limitação de usar o Number.parseInt(n) é que ele não recebe números reais, ou seja, se você utilizar por exemplo 7.5 + 5 ele vai retornar 12 e não 12.5. Para isso podemos utilizar o Number.parseFloat(n). Porém existe outra maneira de realizar essa adição utilizando somente o Number(n) (mas atenção, ele não descarta as outras maneiras realizar).

Utilizando somente o Number(n) ficará: Number(window.prompt('Digite um número: '))

Outra maneira de manipular um número:

Declaramos a seguinte variável: var n1 = 1543.5

Podemos usar a sintaxe n1.toFixed(2) para transformar o número para 1543.50

E caso queira trocar o . para , podemos usar n1.toFixed(2).replace('.', ',')

E para formatar Strings?

Para usar a conversão de número para string, existem duas maneiras:

String(n)

n.toString()

Declaramos a seguitne variável: var s = 'JavaScript'

'Eu estou aprendendo s' // não faz interpolação

'Eu estou aprendendo' + s // usa concatenação

`Eu estou aprendendo ${s}` // usa template string

s.lenght // quantos caracteres a string tem

s.toUpperCase() // tudo para MAIÚSCULAS

s.toLowerCase() // tudo para minúsculas
