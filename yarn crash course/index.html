Aprendendo comandos básicos de Yarn

O Yarn é um gerenciador de pacotes do JavaScript criado pelo Facebook, com sintaxe e funcionalidade similar ao NPM.

Para instalar o yarn é preciso fazer o download no site, apenas procure por yarn download no google.

Conferindo a versão do yarn, usamos yarn -v

Também temos o comando yarn help quie nos dará uma lista de comandos de ajuda com o yarn.

Também temos o comando yarn init que é praticamente a mesma sintaxe do npm init, também podemos usar yarn init -y para criar um package.json com as configurações padrão.
