Nessa aula falaremos sobre operadores relacionais, lógicos e ternário.

Operadores relacionais

Os operadores relacionais são os mesmos da maioria das linguagens de programação, sao eles:

  • 5 > 2 // 5 é maior que 2 resultando em (TRUE)
  • 7 < 4 // 7 não é menor que 4, resultando em (FALSE)
  • 8 >= 8 // 8 é maior ou igual a 8, resultando em (TRUE)
  • 9 <= 7 // 9 não é menor ou igual a 7, resultando em (FALSE)
  • 5 == 5 // 5 é igual a 5, resultando em (TRUE)
  • 4 != 4 // 4 não é diferente de 4, resultando em (FALSE)

Exemplos:

preço >= 200.50 // o preço é maior ou igual a 200.50?

idade < 18 // a idade é menor do que 18?

curso == 'JavaScript' // o curso é JavaScript?

n1 != n2 // o primeiro número é diferente do segundo?

Operadores relacionais de identidade:

Operadores relacionais de identidade são para conferir se os valores são de mesmo tipo.

  • 5 == 5 -> TRUE // valor 5 do tipo number e valor 5 do tipo number, são iguais.
  • 5 == '5' -> TRUE // valor 5 do tipo number e valor 5 do tipo string, são iguais.
  • 5 === '5' -> FALSE // valor 5 do tipo number e valor 5 do tipo string, não são identicos
  • 5 === 5 -> TRUE // valor 5 do tipo number e valor 5 do tipo number, são identicos

Operadores lógicos

Os operadores lógicos no JavaScript são 3:

  • ! negação
  • && conjunção
  • || disjunção

Operador de negação ! -> ele é tratado como um operador unário, que só tem um operando, depois da exclamação ou é verdadeiro ou é falso.

Operador de conjunção && -> ele é tratado como um operador binário, que tem dois valores lógicos, um de cada lado. O resultado desse operador vai ser verdadeiro somente se os dois valores forem verdadeiros:

  • true && true -> true
  • true && false -> false
  • false && true -> false
  • false && false -> false

Operador de disjunção || -> também é um operador binário de dois valores lógicos, e basta que um deles seja verdadeiro, para o resultado ser verdadeiro:

  • true || true -> true
  • true || false -> true
  • false || true -> true
  • false || false -> false

OBS.: quando temos operadores aritméticos, relacionais e lógicos na mesma expressão, a ordem que é realizada a consulta é a seguinte: aritméticos -> relacionais -> lógicos. Quando a expressão tiver um &&, um || e um ! a ordem de execução é SEMPRE essa -> ! depois && depois ||

Exemplos:

  • idade >= 15 && idade <= 17 // a idade está entre 15 e 17?
  • estado == 'RJ' || estado == 'SP' // o estado é RJ ou SP?
  • salário > 1500 && sexo != 'M' // o salário é acima de 1500 e não é homem?

Ordem de precedência:

  1. Operadores aritméticos: ( ) ** / ...
  2. Operadores relacionais: > < >= ...
  3. Operadores lógicos: ! && ||

Operador ternário

O operador ternário é composto por dois símbolos que aparecem na mesma expressão, a ? e :, e ele se chama ternário pois tem três partes, sendo a sintaxe:

teste ? true : false

Lê-se: teste lógico ? o que acontece quando o teste é verdadeiro : o que acontece quando o teste é falso

Exemplo:

média >= 7.0 ? "Aprovado" : "Reprovado"

Se a média for maior ou igual a 7, ele será aprovado, se a média for menor que 7, ele será reprovado.
